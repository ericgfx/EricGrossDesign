Analysis & Planing

Before we start any deep diving, we need to make sure all the equipment is functioning the target has been well identified. Without this big picutre your setting yourself up for a world of hurt.

Design & Development

After you've identified your intended target, you can begin to design the backend architecture to start gathering key data points. From there you can start developing and testing. At this phase we also like to step back and apply an antagonist thinking. This spins the creative juices.
